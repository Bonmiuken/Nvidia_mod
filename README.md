<h1 align="center">MOD</h1>

Description
This is a modified version of the NVIDIA 470.xx driver installer (.run) specifically patched for modern Linux environments.

Features
Kernel Support: Patched to support Linux Kernel 6.8 and newer.

OS Compatibility: Fixed for Ubuntu 25.04 (Plucky Puffin) and other recent rolling release distributions.

Legacy Support: Keeps NVIDIA Kepler-based GPUs (and others supported by the 470 series) alive on the latest software stacks.
